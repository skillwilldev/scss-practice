Dark Mode — ბონუს დავალება

ღილაკი მხოლოდ .dark-mode კლასს ამატებს body-ზე. შენი SCSS პასუხისმგებელია ყველაფრის შეცვლაზე.

მინიშნება: SCSS-ში გამოიყენე:

body.dark-mode {
  $bg-color:   #111111;
  $text-color: #f0f0f0;
  // ან CSS custom properties-ით:
  --bg:   #111111;
  --text: #f0f0f0;
}
